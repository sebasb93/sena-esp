Anidar Elementos ( > )

Ejemplo: ul>li>a>img
<ul> <li><a href=""><img src="" alt=""></a></li> </ul>

Elementos Hermanos ( + )

Ejemplo: header>div+nav
<header> <div></div> <nav></nav> </header>

Repetir Elementos ( + )

Ejemplo: ul>li*5
<ul> <li></li> <li></li> <li></li> <li></li> <li></li> </ul>

Atributo Clase ( + )

Ejemplo: div.contenedor
<div class="contenedor"></div>

Multiples clases ( + )

Ejemplo: div.clase1.clase2.clase3
<div class="clase1 clase2 clase3"></div>

Atributo identificador ( + )

Ejemplo: nav#opciones
<nav id="opciones"></nav>

Adicionar Texto ( + )

Ejemplo: div{texto de prueba}
<div>texto de prueba</div>

Personalizar atributos ( + )

Ejemplo: a[href=www.google.com]
<a href="www.google.com"></a>

Numeros Consecutivos ( + )

Ejemplo: ul>li.con$*4
<ul> <li class="con1"></li> <li class="con2"></li> <li class="con3"></li> <li class="con4"></li> </ul>

Subir de nivel ( + )

Ejemplo: section>article>div^h1
<section> <article> <div></div> </article> <h1></h1> </section>

Agrupar Elementos ( + )

Ejemplo: (div>h1)(div>h2)
<div> <h1></h1> </div> <div> <h2></h2> </div>
